# Product User Guide

## What CourseFlow Helps With

CourseFlow is designed to help Iowa State students and advisors:

- understand degree requirements
- build semester-by-semester plans
- explore majors, courses, and professors
- keep track of current classes
- manage profile and social features inside the app

## Main Product Areas

### Plan

These pages are for long-term academic planning.

#### Flowchart Dashboard

Use this page to:

- manage your flowcharts
- inspect semester layouts
- add or move courses
- review degree progress and planning insights

This is the main planning workspace.

#### Smart Scheduler

Use this page to:

- generate draft semester schedule options
- compare planning outcomes
- explore options before editing a flowchart manually

#### Course Catalog

Use this page to:

- search courses
- browse catalog results
- evaluate whether a course fits a target semester

#### Majors Browse

Use this page to:

- browse imported majors
- inspect requirement groupings
- understand what a major expects before committing to it

### Current Semester

#### Current Classes

Use this page to:

- review the imported current schedule
- see a weekly calendar
- understand active class timing for the current term

### Explore

#### Course Reviews

Use this page to:

- browse course reviews
- search by course
- read student feedback on workload, difficulty, and experience

#### Professor Reviews

Use this page to:

- browse professor profiles
- compare teaching feedback
- review internal and imported external signals when available

### Profile & Social

#### Profile

Use this page to:

- update identity details
- manage profile presentation
- maintain account-facing personal information

#### Student Search / Friends

Use these features to:

- find other students
- view profiles
- build social connections inside the app

#### Course Badges

Use this page to:

- see completed badges
- review next achievable badges tied to in-progress coursework

#### Settings

Use this page to:

- adjust app preferences
- manage module visibility and personal configuration

### More

These are helpful but less central to academic planning:

- Dining
- Games
- Canvas shortcut
- Campus map shortcut

## Typical Student Workflow

1. Create an account
2. Select a major
3. Open the Flowchart Dashboard
4. Import or build a semester plan
5. Use Majors Browse and Course Catalog to fill gaps
6. Use Course and Professor Reviews before choosing classes
7. Track the active semester in Current Classes

## Typical Advisor Workflow

1. Sign in as advisor/faculty
2. Review student plans and flowcharts
3. Use majors, catalog, and review tools to discuss options
4. Help students understand progress and requirement coverage

## What To Check First If Something Looks Wrong

- missing major dropdown options usually means major data is not seeded
- blank review pages may mean imported data is missing or the DB is empty
- current-class issues may come from missing or malformed imported schedule data
- if the app loads but data is empty, confirm which database mode is active
